### 使用框架

ui框架：vant
* svg: svg-sprite-loader config文件也要配置
* 滚动：cnpm i better-scroll -S
* svg包装 npm install svg-sprite-loader -D
* 本地存储 npm install vue-ls --save
* 工具库  cnpm i lodash -S
* 移动端自适应插件 npm install postcss-px-to-viewport --save
* http npm install --save axios
